# Sistema de Gestión, Simulación y Análisis de Pruebas Saber Pro

Es una plataforma digital institucional diseñada para centralizar y administrar todo lo relacionado con las pruebas de Estado Saber Pro. Su propósito principal es permitir la carga y el almacenamiento seguro de los resultados de los exámenes para realizar diagnósticos sobre el rendimiento de los estudiantes. A partir de esta información, el sistema facilita la planificación, el seguimiento y la evaluación de actividades pedagógicas de refuerzo, apoyando así a los directivos y docentes en el diseño de estrategias orientadas a la mejora continua de la calidad académica en la universidad.

## Tecnologías Utilizadas

* **Lenguaje de Programación:** Java
* **Interfaz Gráfica:** JavaFX (FXML)
* **Arquitectura:** Arquitectura Limpia (Clean Architecture)
* **Paradigma:** Programación Orientada a Objetos
